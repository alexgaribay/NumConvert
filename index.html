Welcome to NumConvert

NumConvert is an Android app that converts numbers between different bases. All conversions are done using standard Java libraries. The program is currently being refactored to use a MVC design pattern. Design was not really in mind when originally developing the app.

Authors

Alex Garibay (@alexgaribay) created this app in 2009 and made it open source in 2012.
